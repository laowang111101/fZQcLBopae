# 项目概述 java1195

本项目是一个基于 Java + SpringBoot + Vue 技术栈的音乐平台管理系统，采用前后端分离的架构模式，实现了用户、游客及管理员三大模块的功能。

# 技术栈

- 架构：B/S、MVC
- 开发环境：IDEA、JDK1.8、Maven、Mysql
- 技术支持：Java、Mysql、SpringBoot、Mybatis、Vue

# 功能模块

## 用户模块

- 用户注册
- 用户登录

## 游客模块

- 歌曲播放器
- 歌手列表
- 歌单列表
- 我的音乐
- 歌曲播放
- 歌曲收藏
- 歌曲下载
- 歌曲评论

## 管理员模块

- 用户管理
- 歌手管理
- 歌单管理
- 上传歌曲

## 特殊功能

- 支持音乐歌单推荐（基于协同过滤算法实现）

# 系统角色

- 用户
- 游客
- 管理员

# 运行环境

- 系统环境：Windows
